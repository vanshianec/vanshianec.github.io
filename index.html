Database Management System Documentation

Introduction

Database Management System is a C++ program which can be used to store data in a binary file in the format of tables which can contain columns with data values. The following functionality is supported:

  • Tables can be created which are stored in a binary file
  • Each table contains columns with data in the form of character sequence (string), boolean type (bool), integer (int) or floating point numbers (double)
  • Data in the tables can be inserted, updated or selected
  • One of the columns can be set as 'Primary' which means that only unique values can be inserted in that specific column

Getting Started

To use the system you need to have a C++ compiler installed on your operating system. After that you can clone the project folder on your local machine. Then you need to compile all .cpp files in the project folder and the program will launch and wait for a user input from the console.

How to use the system

Create Table Command

When the user launches the system for the first time he needs to create a table to work on. The command used to create a table is the following:

CREATE TABLE <table_name> (<column1> <datatype1>, <column2> <datatype2>, …) [PRIMARY KEY (<ID>)]
  • "<table_name>" : The name of the table
  • "<column<i>>" : The name of the i-th column of the table which can contain only alphabetic characters and underscore
  • "<datatype<i>>" : The type of the i-th column of the table which can be a character sequence, boolean type, integer or floating point number
  • "<ID>" : (OPTIONAL) The name of the column which will be set as a primary column

Example:

CREATE TABLE myTableName (myFirstColumn int, mySecondColumn double, myThirdColumn bool, myForthColumn string) PRIMARY KEY (<myFirstColumn>);

Insert Command

After that the user can insert values in the table with the following command:

INSERT INTO <table_name> (<column1>, <column2>, ...)
  • "<table_name>" : The name of the table
  • "<column<i>>" : The value which will be inserted in the i-th column of the table (The type of the column must match the given value to be inserted)

Example:

INSERT INTO myTableName (1, 2,99, true, "some string");

Update Command

After we have some values in the table we can use other commands like the update command:

UPDATE <table_name> SET <column1> = <value1>, <column2 = value2>, … WHERE <key> <operator> <value>;
  • "<table_name>" : The name of the table
  • "<column<i>>" : The name of the i-th column from which the values will be updated
  • "<value<i>>" : The new value to be set for the i-th column
  • "<key>" : The name of the column by which the rows will be filtered
  • "<value>" : The value of the column by which the rows will be filtered
  • "<operator>" : The operator which will be applied to <key> and
  • Supported operators are: '!=', '=', '>', '<', '<=', '>='

Example:

UPDATE myTableName SET myFirstColumn = 2, mySecondColumn = 3.00 WHERE myThirdColumn = true;

Select Command

We can also select specific values from the table which will be showed on the console:

SELECT <column1>, <column2>, … FROM <table_name> [WHERE <key> <operator> <value>];
  • "<table_name>" : The name of the table
  • "<column<i>>" : The name of the i-th column from which the values will be updated
  • "<key>" : The name of the column by which the rows will be filtered
  • "<value>" : The value of the column by which the rows will be filtered
  • "<operator>" : The operator which will be applied to <key> and
  • Supported operator are: '!=', '=', '>', '<', '<=', '>='
  • [WHERE ...] is optional

We can also select all table columns with the '*' character:

SELECT * FROM <table_name> [WHERE <key> <operator> <value>];

Example:

SELECT myFirstColumn, mySecondColumn FROM myTableName WHERE myFirstColumn > 1;

We can even order the selected data by a column:

SELECT … ORDER BY <column> ASC|DESC;

Example:

SELECT * FROM myTableName ORDER BY myFirstColumn ASC;

The system supports aggregate functions as well:

SELECT <aggregate1>(<column1>), <aggregate2>(<column2>), … FROM <table_name> [WHERE …];
  • "<table_name>" : The name of the table
  • "<column<i>>" : The name of the i-th column from which the values will be aggregated
  • "<aggregate<i>" : The type of aggregate to be used on the i-th column. Supported aggregates are MIN, MAX, AVG, SUM, COUNT
  • [WHERE ...] is optional

Example:

SELECT MAX(myFirstColumn), AVG(mySecondColumn), COUNT(myThirdColumn) FROM myTableName WHERE myFirstColumn = 4;

Create Index Command

The columns of the table can be indexed which will increase the speed of which data is retrieved and modified but in an expense of more memory:

CREATE INDEX on <table_name>(<column>);
  • "<table_name>" : The name of the table
  • "<column>" : The column for which the index will be created

Quit Command

To stop the program you can simple type the quit command:

QUIT;
  • You don't need to call the quit command to save your progress in the file. Every time you execute a command the progress is saved automatically
  • After quitting you can simply start the program again and continue at where you left it
